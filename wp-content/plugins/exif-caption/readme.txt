== Description ==

* Insert the Exif data to the caption of the media.
* Also replaced caption of content.

== Installation ==

1. Upload `exif-caption` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
